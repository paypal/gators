    "## 3. Build Feature Engineering Pipeline\n",
    "\n",
    "Create a comprehensive pipeline that demonstrates the power of gators transformers:\n",
    "\n",
    "### Missing Value Indicators:\n",
    "- **IsNull**: Create binary indicators for missing Age and Cabin values\n",
    "\n",
    "### String Feature Engineering:\n",
    "- **Length**: Calculate ticket string length\n",
    "- **SplitExtract**: Extract passenger title from name (e.g., 'Mr.', 'Mrs.', 'Miss.')\n",
    "- **ExtractSubstring**: Extract cabin deck letter from cabin number\n",
    "\n",
    "### Mathematical Features:\n",
    "- **MathFeatures**: Sum SibSp and Parch to get family size components\n",
    "- **ScalarMathFeatures**: Add 1 to include the passenger (FamilySize = SibSp + Parch + 1)\n",
    "- **MathFeatures**: Calculate fare per person (Fare / FamilySize)\n",
    "\n",
    "### Conditional Features:\n",
    "- **ConditionFeatures**: Create 'IsAlone' indicator for passengers traveling solo\n",
    "\n",
    "### Data Cleaning:\n",
    "- **RenameColumns**: Give intuitive names to extracted features\n",
    "- **DropColumns**: Remove raw columns after feature extraction\n",
    "- **NumericImputer**: Fill missing numeric values with mean\n",
    "- **StringImputer**: Fill missing categorical values with 'MISSING'\n",
    "\n",
    "### Discretization:\n",
    "- **CustomDiscretizer**: Bin ages into meaningful groups (child, teen, adult, senior)\n",
    "\n",
    "### Encoding:\n",
    "- **RareCategoryEncoder**: Group infrequent categories to reduce noise\n",
    "- **CastColumns**: Convert Pclass to string for categorical treatment\n",
    "- **InteractionFeatures**: Create feature interactions (e.g., Pclass × Age group)\n",
    "- **WOEEncoder**: Apply Weight of Evidence encoding for all categorical features"
   ]

    "# Define the feature engineering pipeline\n",
    "steps = [\n",
    "    # Create missing value indicators\n",
    "    ('IsNull', IsNull(subset=['Age', 'Cabin'])),\n",
    "    \n",
    "    # String feature engineering\n",
    "    ('Length', Length(subset=['Ticket'])),\n",
    "    ('SplitExtractName', SplitExtract(subset=['Name'], by=', ', n=1)),\n",
    "    ('SplitExtractTitle', SplitExtract(subset=['Name__split_,__1'], by='.', n=0)),\n",
    "    \n",
    "    # Calculate family size\n",
    "    ('MathFeatures', MathFeatures(\n",
    "        groups=[['SibSp', 'Parch']],\n",
    "        operations=['sum'],\n",
    "        new_column_names=['Dummy']\n",
    "    )),\n",
    "    ('ScalarMathFeatures', ScalarMathFeatures(\n",
    "        operations=[{'column': 'Dummy_sum', 'op': '+', 'scalar': 1}],\n",
    "        new_column_names=[\"FamilySize\"]\n",
    "    )),\n",
    "    \n",
    "    # Extract cabin deck\n",
    "    ('ExtractSubstring', ExtractSubstring(subset=['Cabin'], start=0, end=1)),\n",
    "    \n",
    "    # Rename for clarity\n",
    "    ('RenameColumns', RenameColumns(column_mapping={\n",
    "        'Name__split_,__1__split_._0': 'Title',\n",
    "        'Cabin__start0_end1': 'CabinDeck'\n",
    "    })),\n",
    "    \n",
    "    # Handle rare categories\n",
    "    ('RareCategoryEncoder', RareCategoryEncoder()),\n",
    "    \n",
    "    # Calculate fare per person\n",
    "    ('MathFeatures2', MathFeatures(\n",
    "        groups=[['Fare', 'FamilySize']],\n",
    "        operations=['div'],\n",
    "        new_column_names=['FarePerPerson']\n",
    "    )),\n",
    "    \n",
    "    # Create 'traveling alone' indicator\n",
    "    ('ConditionFeatures', ConditionFeatures(\n",
    "        conditions=[{\"column\": \"FamilySize\", \"op\": \">\", \"value\": 1}],\n",
    "        new_column_names=['IsAlone']\n",
    "    )),\n",
    "    \n",
    "    # Drop raw columns\n",
    "    ('DropColumns', DropColumns(subset=['Cabin', 'Ticket', 'Dummy_sum'])),\n",
    "    \n",
    "    # Impute missing values\n",
    "    ('NumericImputer', NumericImputer(strategy='mean')),\n",
    "    ('StringImputer', StringImputer(strategy='constant', value='MISSING')),\n",
    "    \n",
    "    # Discretize age into bins\n",
    "    ('CustomDiscretizer', CustomDiscretizer(\n",
    "        bins={'Age': [0, 12, 18, 35, 60, 100]},\n",
    "        inplace=True\n",
    "    )),\n",
    "    \n",
    "    # Convert passenger class to categorical\n",
    "    ('CastColumns', CastColumns(subset=[\"Pclass\"], dtype=pl.String)),\n",
    "    \n",
    "    # Create feature interactions\n",
    "    ('InteractionFeatures', InteractionFeatures(\n",
    "        subset=['Pclass', 'Age', 'CabinDeck', 'Embarked']\n",
    "    )),\n",
    "    \n",
    "    # Apply Weight of Evidence encoding\n",
    "    ('WOEEncoder', WOEEncoder()),\n",
    "]\n",
    "\n",
    "# Build and apply the pipeline\n",
    "pipe = Pipeline(steps=steps, verbose=True)\n",
    "X_train_transformed = pipe.fit_transform(X_train, y_train)\n",
    "X_test_transformed = pipe.transform(test)\n",
    "\n",
    "print(f\"\\nOriginal features: {X_train.shape[1]}\")\n",
    "print(f\"Engineered features: {X_train_transformed.shape[1]}\")"
