API Reference#

Complete reference for all Gators transformers, organized by functionality.

Data Cleaning#

Column Operations

  • CastColumns - Cast columns to different data types

  • DropColumns - Drop specified columns

  • RenameColumns - Rename columns

Quality Filters

  • CorrelationFilter - Remove highly correlated features

  • DropConstantColumns - Remove constant columns

  • DropDuplicateColumns - Remove duplicate columns

  • DropDuplicateRows - Remove duplicate rows

  • DropHighNaNRatio - Drop columns with high missing ratio

  • DropLowCardinality - Drop low cardinality columns

  • HighCardinalityFilter - Filter high cardinality features

  • Replace - Replace values in columns

  • VarianceFilter - Remove low variance features

Encoders#

Categorical Encoding

  • BinaryEncoder - Binary encoding for categorical features

  • CatBoostEncoder - CatBoost-style encoding

  • CountEncoder - Frequency-based encoding

  • LeaveOneOutEncoder - Leave-one-out encoding

  • OneHotEncoder - One-hot encoding

  • OrdinalEncoder - Ordinal encoding for ordered categories

  • RareCategoryEncoder - Handle rare categories

  • TargetEncoder - Target mean encoding

  • WOEEncoder - Weight of Evidence encoding

Feature Generation - Numeric#

Mathematical Operations

  • ComparisonFeatures - Compare columns

  • DistanceFeatures - Calculate distance between columns

  • MathFeatures - Mathematical operations between columns

  • RatioFeatures - Calculate ratio between columns

  • PlanRotationFeatures - Rotate features in space

  • PolynomialFeatures - Generate polynomial features

  • ScalarMathFeatures - Scalar mathematical operations

Conditions and Rules

  • ConditionFeatures - Condition-based features

  • IsNull - Generate null indicator features

  • RuleFeatures - Apply business rules

Group Features

  • GroupStatisticsFeatures - Generate group-based statistics

  • GroupScalingFeatures - Generate group-based scaling features

  • GroupLagFeatures - Generate group-based lag features

  • RowStatisticsFeatures - Generate group-based statistics per row

Feature Generation - String#

Text Properties

  • CharacterStatistics - Character-level statistics

  • Length - String length

  • Occurrences - Count substring occurrences

Text Patterns

  • Contains - Check if string contains substring

  • Endswith - Check if string ends with pattern

  • PatternDetector - Detect regex patterns

  • Startswith - Check if string starts with pattern

Text Transformation

  • ExtractSubstring - Extract substring

  • Lower - Convert to lowercase

  • NGram - Generate n-gram

  • Split - Split based on a delimiter

  • SplitExtract - Split based on a delimiter and extract

  • Upper - Convert to uppercase

Text Interaction

  • InteractionFeatures - Exhaustive string interactions

  • CombineFeatures - Selected string interactions

Feature Generation - DateTime#

Temporal Features

  • BusinessTimeFeatures - Business hours/days

  • CyclicFeatures - Cyclical time encoding

  • DiffFeatures - Time differences

  • DurationToDatetime - Convert duration to datetime

  • OrdinalFeatures - Extract time components

  • HolidayFeatures - Holiday indicators

  • TimeBinFeatures - Time binning

  • TimeWindowFeatures - Time-based window aggregation features

Imputers#

Missing Value Imputation

  • BooleanImputer - Impute boolean columns

  • GroupByImputer - Group-based imputation

  • NumericImputer - Impute numeric columns

  • StringImputer - Impute string columns

Discretizers#

Binning Methods

  • CustomDiscretizer - Custom bin edges

  • EqualLengthDiscretizer - Equal-width binning

  • EqualSizeDiscretizer - Equal-frequency binning

  • GeometricDiscretizer - Geometric progression bins

  • QuantileDiscretizer - Quantile-based binning

  • TreeBasedDiscretizer - Decision tree bins

Scalers#

Feature Scaling

  • ArcSinSquareRootScaler - Arcsinh square root transformation

  • ArcSinhScaler - Arcsinh transformation

  • BoxCox - Box-Cox power transformation

  • LogScaler - Log transformation

  • MinmaxScaler - Min-max normalization

  • PowerScaler - Power transformation

  • StandardScaler - Z-score standardization

  • YeoJonhson - Yeo-Johnson power transformation

Pipeline#

  • Pipeline - Chain multiple transformers

Feature Selection#

  • compute_iv() - Compute Information Value (IV) for features

  • feature_stability_index() - Measure feature importance stability
